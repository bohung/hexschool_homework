CSS 教學

在這堂課程上面，你瞭解如何使用 CSS 設定用在 HTML 標籤上，以及設定 連結顏色

這章節你認識的 CSS 設定文字語法

  1. 文字顏色：color
  2. 文字大小：font-size
  3. 文字行距：line-height
  4. 文字字型：font-family
  5. 其他...

下列的 emmet 語法，哪一項是錯的？

  • ! = HTML5 環境
  • link:css = 匯入 css
  • p*4 = 顯示 4 個 p 段落
  • h1 = 展開 h1 的開頭與結尾的 html 標籤
  • lorem = 自動幫我們寫完網頁
